# Installation

!!! warning "Pre-release"
    repoman is not yet publicly available. This page will be updated with final installation instructions at launch.

## Requirements

- Ubuntu 24.04 LTS or later (any official flavor)
- GTK4 and libadwaita 1.4 (included — no extra packages needed)

## Via PPA (recommended)

```bash
sudo add-apt-repository ppa:tecktron-studios/repoman
sudo apt update && sudo apt install repoman
```

## Via .deb package

Download the latest `.deb` from the [GitHub releases page](https://github.com/Tecktron/repoman/releases) and install it:

```bash
sudo dpkg -i repoman_*.deb
```

## Building from source

See the [README](https://github.com/Tecktron/repoman) for build instructions.
